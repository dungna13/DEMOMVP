# CleanBot System

## Project Overview
CleanBot is an automated system designed to handle cleaning and summarization tasks. It operates on a continuous loop, processing commands from a queue and executing them asynchronously.

## Installation and Usage

### Prerequisites
*   Python 3.8 or higher
*   pip (Python package installer)

### Running the Bot (Easy Way - Windows)
Simply double-click the **`run_bot.bat`** file in the project folder.

### Manual Setup (For Developers)
1.  Install the required dependencies:
    ```bash
    pip install -r auto_bot/requirements.txt
    ```

2.  Run the application:
    ```bash
    python auto_bot/main.py
    ```
